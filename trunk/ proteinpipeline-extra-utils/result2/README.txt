This folder contains results obtained by running 
post-processing module of ProteoAnnotator.  The 
*-mapped.txt files are the final files for Circos plots.
They are generated from *-filtered.txt + GFF combintation.
The *-filtered.txt are obtained by all-PSM-genemode.txt files
by removing the redundant rows. The all-PSM-genemode.txt files
are produced by running the code in analysis2 folder on all-PSM.txt
file which is produced by using cut command in a shall script.
